# Configuration

Thư mục chứa các file cấu hình cho ứng dụng.

## Mục đích
- Cấu hình môi trường (development, production)
- Cấu hình third-party libraries
- App settings

## Ví dụ cấu trúc
```
config/
├── index.js
├── env.js             # Environment variables
├── theme.js           # Theme configuration
└── app.config.js      # General app config
```
