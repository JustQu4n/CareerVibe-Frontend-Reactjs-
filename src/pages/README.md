# Pages

Thư mục chứa các page components (tương ứng với routes).

## Mục đích
- Mỗi file là một trang/route trong ứng dụng
- Kết hợp nhiều components nhỏ
- Container components

## Ví dụ cấu trúc
```
pages/
├── Home/
│   ├── index.jsx
│   └── Home.module.css
├── Jobs/
│   ├── index.jsx
│   ├── JobList.jsx
│   └── JobDetail.jsx
├── Auth/
│   ├── Login.jsx
│   └── Register.jsx
├── Profile/
│   └── index.jsx
└── NotFound.jsx
```
