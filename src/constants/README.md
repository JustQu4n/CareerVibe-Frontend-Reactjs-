# Constants

Thư mục chứa các hằng số được sử dụng trong toàn bộ ứng dụng.

## Mục đích
- Định nghĩa các giá trị không thay đổi
- Tránh hardcode trong code
- Dễ dàng bảo trì và cập nhật

## Ví dụ cấu trúc
```
constants/
├── index.js
├── routes.js          # Route paths
├── apiEndpoints.js    # API URLs
├── status.js          # Status codes, states
├── roles.js           # User roles
└── messages.js        # Error/Success messages
```
