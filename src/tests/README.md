# Tests

Thư mục chứa test files và test utilities.

## Mục đích
- Unit tests
- Integration tests
- Test utilities và mocks
- Test configuration

## Ví dụ cấu trúc
```
tests/
├── unit/
│   ├── components/
│   ├── hooks/
│   └── services/
├── integration/
├── e2e/
├── mocks/
│   ├── mockData.js
│   └── mockAPI.js
└── utils/
    └── testHelpers.js
```

## Testing Libraries
- Jest
- React Testing Library
- Vitest (nếu dùng Vite)
