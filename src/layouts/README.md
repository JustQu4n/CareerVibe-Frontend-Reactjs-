# Layouts

Thư mục chứa các layout components cho các trang khác nhau.

## Mục đích
- Tạo cấu trúc chung cho nhiều pages
- Header, Footer, Sidebar components
- Layout cho các roles khác nhau

## Ví dụ cấu trúc
```
layouts/
├── MainLayout.jsx        # Layout chính cho user
├── AdminLayout.jsx       # Layout cho admin
├── AuthLayout.jsx        # Layout cho login/register
├── DashboardLayout.jsx
└── components/
    ├── Header.jsx
    ├── Footer.jsx
    └── Sidebar.jsx
```
