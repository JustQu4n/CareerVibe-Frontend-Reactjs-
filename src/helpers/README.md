# Helpers

Thư mục chứa các helper/utility functions.

## Mục đích
- Pure functions để xử lý data
- Format, parse, transform functions
- Common utilities

## Ví dụ cấu trúc
```
helpers/
├── dateHelper.js      # Format dates
├── stringHelper.js    # String manipulation
├── numberHelper.js    # Number formatting
├── arrayHelper.js     # Array operations
├── validationHelper.js
└── storageHelper.js   # LocalStorage/SessionStorage
```
