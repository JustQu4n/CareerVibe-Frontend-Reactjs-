# Services Layer

Thư mục chứa business logic và xử lý dữ liệu từ API.

## Mục đích
- Xử lý logic nghiệp vụ
- Transform data từ API
- Tái sử dụng logic giữa các components

## Ví dụ cấu trúc
```
services/
├── authService.js
├── jobService.js
├── companyService.js
├── applicationService.js
└── storageService.js
```
